Die Webseite ist eine HTML-Datei und ein CSS-Datei. Sie sollen im gleichem Ordner sein. Das dazugehörige Bild gehört in den gleichen Order wie die 2 Dateien.

Um die Webseite anschauen zu können, kann man den Ordner im XAMPP-Ordner htdocs speichern. Laufwerk:\Ordner\xampp\htdocs
Alternative Method: Visual Studio Code benutzen, Erweiterung 'Live-Server' herunterladen. Die Anleitung wie die Erweiterung funktioniert wird im Visual Studio Code bei der Erweiterung gezeigt.
